== Running locally

.Start HealthID API Server
---
cd healthid/api
source venv.sh
uvicorn healthid:app --reload
---

.Start UI Server
---
cd healthid/vue
yarn serve
---


.Create Docker image
---
cd healthid
docker build . -t healthid
---

.Run Docker image
---
cd healthid
docker run -it -p 80:80 --rm healthid 
---
